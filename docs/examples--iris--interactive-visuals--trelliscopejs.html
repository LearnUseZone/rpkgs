Source file: code-rmd/examples/iris/interactive-visuals/trelliscopejs.Rmd

Interactive scatter plots with regression lines

These scatter plots are created also by using trelliscopejs::facet_trelliscope() which has advantages but also few disadvantages.

One of advantages is a possibility to manage displaying plots using the left pane.

One of disadvantages is a warning message that pops-up after using as_plotly = TRUE.

  • More about this warning is here.
  • It is also reflected in a created HTML file in a result text informing that some file couldn’t be loaded. JavaScript is used in the source .Rmd file to hide this unwanted message.

More about trelliscopejs is here and in README file.

knitr::read_chunk("code/examples/iris/interactive-visuals/trelliscopejs.R")

# scatter plots with regression lines using trelliscopejs
iris_long %>%
  ggplot(aes(x = `x`, y = `y`)) +
  geom_point(color = "black", aes(shape = factor(1))) +
  scale_shape_manual(values = 1) +
  geom_vline(xintercept = 0:8, color = "white", size = 0.25) +
  geom_hline(yintercept = 0:8, color = "white", size = 0.25) +
  geom_smooth(method = "lm", formula = "y ~ x", se = F, color = "orange") +
  geom_ribbon(method = "lm", formula = "y ~ x", stat = "smooth", fill = "orange", alpha = .4) +
  theme(legend.position = "none", axis.ticks = element_blank()) +
  trelliscopejs::facet_trelliscope(
    . ~ Species + `x ` + `y `,
    nrow = 1,
    ncol = 3,
    scales = c("same", "same"),
    name = "Regressions",
    path = "trelliscopejs/regressions",
    as_plotly = T,  # cause warning: https://github.com/hafen/trelliscopejs/issues/61
    data = iris_long
  )
