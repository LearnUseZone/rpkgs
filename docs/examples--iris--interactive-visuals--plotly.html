Interactive scatter plot with qualitative colorscales

Interactive scatter plot where three species of Iris (Iris setosa, Iris virginica and Iris versicolor) are distinguished by color.

Load prepared data

knitr::read_chunk("code/examples/iris/interactive-visuals/plotly.R")

Scatter plot

# qualitative color scales
plotly::plot_ly(
  data = iris,
  x = ~Sepal.Length,
  y = ~Petal.Length,
  mode = "markers",
  type = "scatter",
  color = ~Species,
  colors = "Set2"
)
